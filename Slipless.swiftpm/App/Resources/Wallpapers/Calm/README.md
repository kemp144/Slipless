# Calm Wallpaper Set (iPhone 13+)

This folder contains calm-style wallpapers tuned for iPhone aspect classes.

Both source SVG and ready-to-use PNG files are included.

## Files

- `calm-1080x2340.(svg|png)` for compact 5.4" class (iPhone 13 mini)
- `calm-1170x2532.(svg|png)` for 6.1" legacy class (iPhone 13, iPhone 13 Pro, iPhone 14)
- `calm-1179x2556.(svg|png)` for 6.1" modern class (iPhone 15/16/17 base class)
- `calm-1206x2622.(svg|png)` for 6.3" Pro class (iPhone 16 Pro/17 Pro class)
- `calm-1290x2796.(svg|png)` for 6.9" Plus/Max class
- `calm-1320x2868.(svg|png)` for high-density 6.9" Pro Max class (iPhone 17 Pro Max class)

## Device guidance

Use the file with the closest native resolution for the target device.

- iPhone 13 mini: `calm-1080x2340.png`
- iPhone 13 / 13 Pro / 14: `calm-1170x2532.png`
- iPhone 13 Pro Max / 14 Plus: `calm-1290x2796.png`
- iPhone 14 Pro: `calm-1179x2556.png`
- iPhone 14 Pro Max: `calm-1290x2796.png`
- iPhone 15 / 15 Pro / 16 / 17: `calm-1179x2556.png`
- iPhone 15 Plus / 15 Pro Max / 16 Plus / 16 Pro Max: `calm-1290x2796.png`
- iPhone 16 Pro / 17 Pro: `calm-1206x2622.png`
- iPhone 17 Pro Max: `calm-1320x2868.png`

## Xcode preview tip

Use `Image("calm-1179x2556")` in SwiftUI previews when you need to inspect the wallpaper directly.
